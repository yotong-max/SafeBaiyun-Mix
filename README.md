
# 平安回家

本软件是广州市白云区蓝牙门禁的离线版本，只需门禁的 MAC 地址以及加密 Key 即可开门，无需网络连接。

## 功能特性

- 🚪 **离线开门** - 无需网络，本地蓝牙连接即可开门
- 📱 **多门禁管理** - 支持添加和管理多个门禁设备
- 🎯 **快捷操作** - 桌面小部件支持（中/大/特大尺寸）
- 📲 **二维码导出导入** - 一键导出/导入门禁配置，方便多设备同步
- 🔧 **自动构建发布** - 通过 GitHub Actions 自动构建并发布 APK 到 Releases

## 使用方法

1. 提取 MAC 地址及加密密钥，[获取方式](extract.md)
2. 点击软件右上角编辑按钮，将 MAC 地址及 Key 填进去，保存
3. 点击开门按钮即可开门

## 二维码同步功能

- **导出配置**：在主界面菜单中选择「导出配置」，生成包含所有门禁配置的二维码，可保存到相册或分享给他人
- **导入配置**：在主界面菜单中选择「扫描导入」，扫描二维码即可一键导入门禁配置，支持「替换全部」或「追加到现有」两种模式

## 下载

在 [Releases](https://github.com/yotong-max/SafeBaiyun-Mix/releases) 页面下载最新的 APK 安装包：
- **app-release.apk** - 已签名的发布版本（推荐安装）
- **app-debug.apk** - 调试版本

## 构建与发布

本项目使用 GitHub Actions 进行自动构建和发布：
- 推送到 `main` 分支或创建 PR 时自动构建 APK
- 推送 `v*` 格式标签时自动构建并发布到 GitHub Releases
